# 🤖 FinAI - Assistente Financeiro Inteligente

## 📌 Sobre o Projeto

O FinAI é um assistente financeiro desenvolvido com foco em educação financeira e apoio na tomada de decisão.

Ele utiliza conceitos de IA para responder dúvidas, simular empréstimos e orientar o usuário de forma simples e acessível.

---

## 🚀 Funcionalidades

- 💬 Chat interativo
- 💰 Simulação de empréstimos
- 📊 Explicação de conceitos financeiros
- 🧠 Condução de conversa com contexto

---

## 🛠️ Tecnologias

- Python
- Streamlit

---

## ▶️ Como executar

### 1. Instale as dependências:

```bash
pip install streamlit

Execute o projeto:
python -m streamlit run src/app.py

Acesse no navegador:
